## 图片规格建议

- **格式**：PNG（推荐，支持透明背景）或 JPG
- **尺寸**：宽度 1920px，高度根据内容调整（建议 1080px-1440px）
- **文件大小**：建议压缩后小于 500KB
- **命名**：使用小写字母和连字符，如 `kitfoxpay-admin.png`

## 获取图片的方式

### 方式一：实际截图（推荐）

按照 `占位图生成说明.md` 中的指引，在相应系统中截图。

### 方式二：使用占位图

如果暂时没有实际截图，可以使用 `占位图生成说明.md` 中提供的占位图链接。

### 方式三：使用设计工具制作

使用 Figma、Canva 等工具制作示意图。

## 注意事项

1. **路径问题**：
   - 在 `docs/` 目录下的文档中使用：`./images/xxx.png`
   - 在根目录的 `README.md` 中使用：`docs/images/xxx.png`

2. **GitHub 显示**：
   - 确保图片已提交到 Git 仓库
   - GitHub 会自动显示仓库中的图片

3. **外部平台发布**：
   - 如果要在 CSDN、知乎等平台发布，需要将图片上传到平台图床
   - 或使用 GitHub 的 raw 链接：`https://raw.githubusercontent.com/kitfoxai/kitfoxpay/main/docs/images/xxx.png`
