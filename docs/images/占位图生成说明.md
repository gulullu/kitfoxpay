## 方式二：使用实际截图

### 截图建议

1. **KitfoxPay 管理界面** (`kitfoxpay-admin.png`)
   - 截图位置：访问 `http://localhost:9219` 后的配置管理界面
   - 建议包含：Jeepay 配置、易支付接口配置、服务器配置等区域

2. **KitfoxPay 调试界面** (`kitfoxpay-debug.png`)
   - 截图位置：管理界面中的测试/调试功能区域
   - 建议包含：接口测试、日志查看等功能

3. **NewAPI 配置界面** (`newapi-config.png`)
   - 截图位置：NewAPI 系统中的支付配置页面
   - 建议包含：epay_url、pid、key 等配置项

4. **NewAPI 支付界面** (`newapi-payment.png`)
   - 截图位置：用户在 NewAPI 系统中发起支付的页面
   - 建议包含：支付方式选择、金额显示等

5. **NewAPI 支付成功界面** (`newapi-payment-success.png`)
   - 截图位置：支付成功后的提示页面
   - 建议包含：成功提示、订单信息等

6. **Jeepay 商户信息** (`jeepay-merchant-info.png`)
   - 截图位置：Jeepay 后台的商户信息页面
   - 建议包含：商户号、应用ID等信息的显示位置

7. **Jeepay API 密钥** (`jeepay-api-key.png`)
   - 截图位置：Jeepay 后台的 API 密钥管理页面
   - 建议包含：商户私钥的获取位置

### 截图工具推荐

- Windows: Snipping Tool、ShareX
- macOS: Command + Shift + 4
- Linux: Flameshot、GNOME Screenshot

### 图片处理建议

- 格式：PNG（推荐）或 JPG
- 尺寸：建议宽度 1920px，高度根据内容调整
- 压缩：使用 TinyPNG 等工具压缩，减小文件大小
- 标注：可以在图片上添加箭头、文字说明等（可选）
